## File Naming Convention

```
web/monster_image/{enemy_key}.webp
```

예시:
- `web/monster_image/goblin.webp`
- `web/monster_image/fallen_archangel.webp`

## Recommended Settings
- **Size**: 512x512 or 1024x1024
- **Format**: WebP (for web performance)
- **Style**: Consistent dark fantasy painterly style across all images
